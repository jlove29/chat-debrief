# Bookmarklets

This directory contains JavaScript bookmarklets for exporting conversation transcripts from Google Gemini and Google AIM.

## Documentation

- **[User Guide](../docs/bookmarklets.md)** - How to create and use bookmarklets
- **[Technical Documentation](../docs/eng/bookmarklets.md)** - Implementation details and debugging

## Quick Start

```bash
# Create a universal bookmarklet
python3 minify.py chat
```

Then follow the instructions in the [User Guide](../docs/bookmarklets.md) to install and use it.
